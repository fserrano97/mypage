Distortit v3.2

Distortit is a simple image morphing tool, that allows you to create funny variations of pictures by applying magnification effects to selected areas. You can select the level of magnification and simply click your mouse to apply the effect. With some imagination and suitable images, you can create some funny effects and save the new image as BMP or JPG file.

To apply more than one effect you must first Keep what you have.  Applying magnification to a large image can take a long time so click Stretch and adjust image size as required then click on Resize.   You can also cut a section out of a large picture using the CTRL key and mouse.  The cut section will replace the current image.

Rotating the image will save the current effects.

NEW v 3.0: A slight improvement in drawing speed. 

               Added a morphing function that uses the first and last thumbnails.  Click the button to start/stop. 

                   Added a basic animation that runs through all thumbnails.  Click button to start/stop.  NOTE  animation will fail if the thumbnails are not all the same size.

NEW v 2.3:  Drop a bitmap or JPEG onto Distortit to load. Also added ability to rotate.

NEW v2.2: Hold down ALT key to distort on moving the mouse.  Don't move the mouse too quickly.

NEW v2.2: Corrected pasting from clipboard.

NEW v2.1:  When you Keep, a thumbnail of the current image is saved so that you can go back to any stage in the development.  Click on a thumb to copy it to the main panel.  Now, between copying to clipboard and using thumbs you can experiment more with distorting an image.

                    To cater better for the new functions, I have moved all options into a pop-up menu rather than using buttons.

NEW v2.0: The drawing method has been changed to increase speed and improve distorting larger images.

Installation:  Extract all files to a folder of your choice.  To uninstall just delete the folder.
